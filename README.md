# Hand-Gesture-Particles-
An interactive 3D simulation using Three.js and MediaPipe Hands. Manipulate a cloud of 10,000 particles through real-time hand tracking.  Morphing: Hand gestures switch shapes between Sphere, Heart, Saturn, and more.  Interaction: Pinch to attract and change colors; open palm to repel.  Tech: WebGL, BufferGeometry.
